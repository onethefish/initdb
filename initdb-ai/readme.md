## 这是一个基于Spring AI Alibaba 智能数据库助手项目

## 功能说明

| 功能点       | 详细说明                               | 状态  |
|-----------|------------------------------------|-----|
| 用户管理      | 多用户使用、用户管理、鉴权等                     | 未开始 |
| 数据源管理     | 记录下已创建过的数据库连接可以在会话创建时直接引用，减少重复创建工作 | 已完成 |
| 会话创建      | 创建一个对话框，可以引用数据源，也可以无数据源对话          | 已完成 |
| 会话删除      | 删除一个对话框（目前只有全部删除）                  | 已完成 |
| 基本问答      | 基本的对话能力                            | 已完成 |
| 流式问答      | 流式输出对话减少用户等待时间                     | 已完成 |
| 思考过程      | 展示大模型的思考过程                         | 进行中 |
| 数据库知识库    | 用户可以管理数据库相关知识文档（RAG）               | 已完成 |
| 数据库所有表查询  | 通过对话完成数据库所有表查询                     | 已完成 |
| 数据库表结构查询  | 通过对话完成数据库表的表结构查询                   | 已完成 |
| 智能生成查询SQL | 通过对话完成查询SQL语句的生成                   | 已完成 |
| 数据库表数据查询  | 通过对话完成数据库表的数据查询（优化展示结果）            | 已完成 |
| 数据库表数据导出  | 通过对话完成数据导出（workflow、多agent协助）      | 未开始 |
| 工程化编译     | 真实可用的工程                            | 已完成 |
